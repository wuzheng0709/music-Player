

## Install

```
$ npm install aplayer --save
```

## CDN

[unpkg](https://unpkg.com/aplayer)

[cdnjs](https://cdnjs.com/libraries/aplayer)

[BootCDN](http://www.bootcdn.cn/aplayer)

## Usage

See [docs](http://aplayer.js.org/docs)

## Run in development

```
$ npm install
$ npm run dev
```

## Make a release

```
$ npm install
$ npm run build
```
